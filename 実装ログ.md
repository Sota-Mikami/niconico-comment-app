# ニコニコ動画風Slackコメントアプリ 実装ログ

実装日: 2026-03-03

## 実装完了ステップ

### Step 1: プロジェクトセットアップ
- `electron-vite` + React + TypeScript 構成を手動セットアップ
- `package.json`, `tsconfig.json`, `electron.vite.config.ts` を作成

**トラブルシュート**:
- Claude Code自体がElectronで動いているため `ELECTRON_RUN_AS_NODE=1` が環境変数に設定されている
- 対策: `npm run dev` に `env -u ELECTRON_RUN_AS_NODE` を付与

### Step 2: ウィンドウ透過・最前面・クリックスルー設定
- `src/main/index.ts`: `transparent: true`, `frame: false`, `hasShadow: false`, `alwaysOnTop: true`
- `setIgnoreMouseEvents(true, { forward: true })` でクリックスルー
- `screen.getPrimaryDisplay().workAreaSize` でフルスクリーン

### Step 3: Slack Socket Mode実装
- `npm install @slack/bolt dotenv`
- `.env` スタブファイル作成（`.env.example` も）
- スタブ値検出ロジック（本番トークン未設定時は警告のみ）

### Step 4 & 5: React コメントアニメーション + レーン管理
- `App.tsx`: IPC受信 + レーン管理アルゴリズム + アニメーション
- レーン管理: 最後にコメントが流れた時刻で「最も空いているレーン」を選択
- CSSアニメーション: `translateX` + `will-change: transform` でGPUアクセラレーション
- `onAnimationEnd` でDOMクリーンアップ

### Step 6: スタイリング
- 日本語フォント指定（Hiragino等）
- 白文字 + 黒縁取り（`text-shadow` 8方向）
- `transform: translateZ(0)` でレンダリング最適化

## 起動方法

```bash
cd 01_mikan/projects/ニコニコ動画風コメントアプリ

# 開発 (デモコメントが自動流れる)
npm run dev

# 本番ビルド
npm run build
```

## Slack接続方法

1. `.env` を開き実際のトークンを設定:
   ```
   SLACK_BOT_TOKEN=xoxb-実際のトークン
   SLACK_APP_TOKEN=xapp-実際のトークン
   SLACK_CHANNEL_ID=C実際のチャンネルID
   ```

2. Slack App設定:
   - Bot Token Scopes: `channels:history`, `channels:read`, `groups:history`
   - Socket Mode を有効化
   - App-Level Token (xapp-) を発行

## Step 7: 設定UI実装（2026-03-03）

### 追加機能
- System Tray アイコン（メニューバー）クリックで設定ウィンドウ開閉
- 複数 Slack チャンネルの GUI 管理（追加・削除）
- 表示スクリーン選択（接続ディスプレイ一覧から選択）
- 設定は `~/Library/Application Support/niconico-comment-app/settings.json` に永続化
- 起動時に `.env` の `SLACK_CHANNEL_ID` を自動移行

### 設定画面の開き方
- メニューバーのアイコンをクリック → 設定ウィンドウが開く

### トラブルシュート
- `preload` が複数ファイルになったため `electron.vite.config.ts` の preload に `rollupOptions.input` を追加
- `renderer` も `settings.html` を追加して multi-page 対応
