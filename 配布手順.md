# Niconico Slack Comment 社内配布手順

## 概要

ニコニコ動画風に Slack のコメントをオーバーレイ表示するアプリです。
`.dmg` ファイルを配布し、受け取った人がインストールするだけで使えます。

---

## ビルド担当者向け: dmg の作り方

### 前提条件

- Node.js 18以上がインストールされていること
- macOS でビルドすること（Windows/Linux では mac 向け dmg は作れません）

### 手順

```bash
# 1. リポジトリをクローン（または最新を pull）
git clone https://github.com/Sota-Mikami/niconico-comment-app.git
cd niconico-comment-app

# 2. 依存パッケージをインストール
npm install

# 3. ビルド & dmg 作成
npm run dist
```

完了すると `dist/` フォルダに以下が生成されます：

```
dist/
├── Niconico Slack Comment-1.0.0-arm64.dmg   ← Apple Silicon (M1/M2/M3) 用
└── Niconico Slack Comment-1.0.0.dmg          ← Intel Mac 用
```

配布先の Mac のチップに合わせて適切な dmg を渡してください。
不明な場合は `arm64` 版を渡すと Apple Silicon Mac では動作します。

## 受け取った人向け: インストール方法

### 1. dmg を開く

ダウンロードした `.dmg` ファイルをダブルクリックして開きます。

### 2. アプリをコピー

`Niconico Slack Comment` アイコンを `Applications` フォルダにドラッグ＆ドロップします。

### 3. 初回起動時の警告を解除

Apple の公証を受けていないため、初回は警告が出ます。
**ターミナルで以下のコマンドを1回実行してください。**

```bash
xattr -cr /Applications/Niconico\ Slack\ Comment.app
```

> ターミナルの開き方: Spotlight（⌘ + Space）→「ターミナル」と入力 → Enter

実行後はダブルクリックで普通に起動できます。

---

## 初期設定

起動するとメニューバー（画面右上）にアプリアイコンが現れます。

**アイコンをクリック → 設定を開く** で設定ウィンドウが開きます。

### 必須設定

#### 1. Slack 接続設定（画面最下部）

| 項目 | 入力値 |
|---|---|
| Bot Token | `xoxb-` から始まるトークン |
| App-Level Token | `xapp-` から始まるトークン |

トークンの取得方法は下記「Slack App の作り方」を参照してください。

#### 2. 監視チャンネルの追加

- チャンネル名で検索 → クリックで追加
- Bot を追加していないチャンネルは表示されません

#### 3. 表示スクリーンの選択

複数モニターがある場合は、コメントを流したいスクリーンをクリックして選択します。

## Slack App の作り方（初回のみ）

1. [api.slack.com/apps](https://api.slack.com/apps) にアクセスして **Create New App → From scratch**
2. App Name と Workspace を入力して作成
3. **Socket Mode を有効化**
   - 左メニュー: Socket Mode → Enable Socket Mode → ON
   - App-Level Token を生成（スコープ: `connections:write`）→ `xapp-...` がコピーできる
4. **Bot Token Scopes を設定**
   - 左メニュー: OAuth & Permissions → Scopes → Bot Token Scopes に以下を追加：
     - `chat:write`
     - `channels:history`
     - `channels:read`
5. **アプリをワークスペースにインストール**
   - OAuth & Permissions → Install to Workspace → Bot User OAuth Token（`xoxb-...`）をコピー
6. **監視チャンネルに Bot を追加**
   - Slack で対象チャンネルを開き、`/invite @アプリ名` と入力

---

## アンインストール

`Applications` フォルダから `Niconico Slack Comment` をゴミ箱に移動するだけです。

設定ファイルも削除する場合：

```
~/Library/Application Support/niconico-comment-app/settings.json
```

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| コメントが流れない | 設定ウィンドウ → デモモードをONにして確認。OFFの場合は Slack Bot の接続状況を確認 |
| 「Apple could not verify…」が出る | ターミナルで `xattr -cr /Applications/Niconico\ Slack\ Comment.app` を実行 |
| チャンネル一覧が取得できない | Bot Token Scopes に `channels:read` が追加されているか確認 |
| フルスクリーンアプリの後ろに隠れる | macOS の仕様で一部アプリでは非表示になる場合があります |
